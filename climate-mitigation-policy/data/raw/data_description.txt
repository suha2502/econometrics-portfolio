Description: 
    This dataset contains information on coloured denim imports to Canada for the years 2000 to 2023, excluding 2017, 
    specifically for the HS code 521142, which refers to coloured denim with less than 85% cotton and greater than 200g/m2.
Columns:
    - Reporter: The country reporting the trade data (in this case, Canada).
    - TradeFlow: The direction of trade, which is "Import" in this dataset.
    - ProductCode: The HS code for the product, 521142 for coloured denim.
    - Product Description: A brief description of the product.
    - Year: The year of the data.
    - Partner: The trade partner country (e.g., China, Italy, United States).
    - Trade Value 1000USD: The value of the imports in 1,000 USD.
    - Quantity: The quantity of denim imported, in kilograms.
    - Quantity Unit: The unit of measure, in this case, kilograms (Kg).
